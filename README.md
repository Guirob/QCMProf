# QCM Prof

QCM Prof is a simple projet to allow school teachers to generate QCM with customizable feedback.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
